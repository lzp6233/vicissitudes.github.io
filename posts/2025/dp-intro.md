## 背景
DP 将问题拆为重叠子问题，通过状态转移方程迭代求解。

## 核心概念
- 状态定义：`dp[i]` 的含义
- 转移方程：`dp[i] = f(dp[i-1], ...)`
- 边界条件：初始化与答案位置

## 示例与代码
```js
// 斐波那契（迭代）
function fib(n){
  if(n<2) return n
  let a=0,b=1
  for(let i=2;i<=n;i++){
    ;[a,b]=[b,a+b]
  }
  return b
}
```

## 常见坑
- 状态含义不清导致转移错误
- 忘记初始化边界条件
